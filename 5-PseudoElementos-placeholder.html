Pseudo Elementos

Usado para adicionar elementos extras na página, sem a necessidade de declará-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::placeholder

Um placeholder é uma dica para ajudar no preenchimento de algum campo do formulário.

Propriedades aceitas:

  • color
  • font
  • backgound
  • text
